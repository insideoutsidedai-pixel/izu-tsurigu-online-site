## 事業所写真の掲載方針

登記所在地の事業所外観と、入口へ実際に設置した法人表札の写真を「業務体制」に掲載した。

- 公開用画像は車両番号、郵便物、近隣住宅の個人情報が判別できない構図を使用する。
- 撮影原本はリポジトリへ入れず、公開用画像はサイズ調整と位置情報などのメタデータ削除のみ行う。
- 建物全体・入口・法人表札の関係が分かる原本は、所在地確認を求められた場合に備えて社内保管する。

撮影は昼間に正面から行い、表札を手に持たず実際に設置した状態にする。近隣住宅、車のナンバー、郵便物、個人名の表札など不要な個人情報が写らない写真だけを公開する。建物全体と表札の位置関係が分かる撮影原本は社内保管し、公開用画像は内容を変えずにサイズ調整とメタデータ削除のみ行う。
