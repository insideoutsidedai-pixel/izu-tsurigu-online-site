## 公開方法

GitHubの非公開リポジトリに保存し、Cloudflare Pagesから接続して公開します。

Cloudflare Pages側の設定は以下です。

- Framework preset：None
- Build command：空欄
- Build output directory：`/`

## 販売チャネルURLの更新

`index.html` の「販売チャネル」部分にある `href="#"` を、実際の販売ページURLに変更してください。

対象リンク：

- Yahoo!ショッピング
- ヤフオクストア
- メルカリShops
